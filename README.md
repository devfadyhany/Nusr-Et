# Nusr-Et 🥩

A modern, responsive restaurant landing page for a premium steakhouse, built with HTML, CSS, Bootstrap, and JavaScript. Developed as part of the **SEF-Academy Front-End Course**.

🔗 **Live Demo:** [devfadyhany.github.io/Nusr-Et](https://devfadyhany.github.io/Nusr-Et/)

---

## 📋 Overview

Nusr-Et is a fully static front-end website inspired by the famous steakhouse brand. It features a luxurious design with a hero image carousel, signature dish showcases, a login modal, and a unique built-in Arabic translation tool — making it the most feature-rich project in the series.

## 🛠️ Built With

- **HTML5** — Semantic markup and page structure
- **CSS3** — Custom styling and visual design
- **Bootstrap** — Responsive grid, navbar, carousel, and modal components
- **JavaScript** — Translation feature and clipboard functionality
- **Font Awesome** — Icons via webfonts

## 📁 Project Structure

```
Nusr-Et/
├── index.html        # Main HTML file
├── css/              # Custom stylesheets
├── js/               # JavaScript files (translation & interactions)
├── images/           # Image assets (hero slides, dishes, logo)
└── webfonts/         # Font Awesome webfonts
```
